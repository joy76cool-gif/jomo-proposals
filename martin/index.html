Three Phases

整體規劃 — 三階段

Phase 1 救火、Phase 2 onboarding、Phase 3 續約。每階段都解一個您親口提到的痛點。
節省時間以「單位節省」呈現，年度總時長等您回 D / G 後可精算。

PHASE 01 · 必做

救火雷達 + 客戶資料底座 + 週報

5–7 月
解決：管理師沒主動關心家庭 → 客訴 / 退費 → 您救火 30-60 分。
同時取代「馬丁手 key Excel」成中央 Sheets 資料庫，建立後續所有自動化的基底。
  • 1.0（6/中）：建資料庫 + 敏感字偵測 + 教練 Line 通知（最快有感）
  • 1.1（6/底）：Escalation 升級 + 紀錄檢查表 + 自動催繳
  • 1.2（7/初）：Looker Studio 週報 + Claude AI 兜底（抓微妙訊號）
30–60 分
每救火事件 → 5 分（看週報）

  • 諮詢前自動生「6 個月變化報告」
  • 馬丁從漏斗最後一段抽身
